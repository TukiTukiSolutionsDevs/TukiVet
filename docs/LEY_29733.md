# Cumplimiento Ley 29733 — TukiVet

Última actualización: 2026-06-02
Reglamento vigente: **DS 016-2024-JUS**
Autoridad: **ANPD** (Autoridad Nacional de Protección de Datos Personales) del MINJUSDH.

---

## 1. Qué cubre TukiVet hoy

### 1.1 Endpoints técnicos implementados

| Derecho ARCO | Endpoint | Implementado |
|---|---|---|
| Acceso | `GET /api/v1/portal/data-export` | ✅ |
| Acceso | `POST /api/v1/portal/data-requests` (`type: "access"`) | ✅ |
| Rectificación | `POST /api/v1/portal/data-requests` (`type: "rectification"`) | ✅ |
| Cancelación | `POST /api/v1/portal/data-requests` (`type: "cancellation"`) | ✅ |
| Oposición | `POST /api/v1/portal/data-requests` (`type: "opposition"`) | ✅ |

### 1.2 Consentimiento

- Tabla `consent` con `body_hash` SHA256 inmutable.
- Endpoint `POST /api/v1/portal/consents` para firma del titular.
- Cada consentimiento queda versionado (`version`) + timestamp + IP + user-agent.

### 1.3 Auditoría

- Tabla `audit_log` registra accesos y modificaciones a datos personales.
- Inmutable: nunca se borra.

### 1.4 Seguridad

- Argon2id para passwords (memory_cost=64MB, time_cost=2).
- JWT con refresh rotation + revocación.
- TLS 1.3 obligatorio en producción (Caddy / reverse proxy).
- Cifrado at-rest del disco del VPS (LUKS o cifrado del proveedor).
- Backups automáticos cifrados (pg_dump + GPG → S3-compatible).

## 2. Lo que el usuario (Jaime) debe completar antes de producción

### 2.1 Registro de banco de datos ante la ANPD

**Obligatorio**: el banco de datos `clientes_y_pacientes_tukivet` debe registrarse en el Registro Nacional de Protección de Datos Personales (RNPDP).

**Procedimiento**: presentar el formulario ante la ANPD con la información de §3 abajo. La constancia de inscripción tiene un código que se debe guardar en `organization.sunat_data_bank_code` (campo ya existente).

**Costo**: gratuito.
**Plazo de evaluación**: ~30 días hábiles.
**Link oficial**: <https://www.gob.pe/autoridad-nacional-de-proteccion-de-datos-personales>

### 2.2 Designar oficial de datos personales

Una persona natural responsable de:
- Atender solicitudes ARCO en máximo **20 días hábiles**.
- Responder a la ANPD ante requerimientos.
- Capacitar al personal.

### 2.3 Política de privacidad publicada

Debe estar disponible en:
- Landing pública: link en footer.
- Portal del cliente: aceptación obligatoria al primer ingreso (consentimiento tipo `data_processing`).

### 2.4 Aviso de tratamiento de datos al recolectar

En el formulario de registro de cliente nuevo (recepción), debe haber:
- Texto del tratamiento.
- Checkbox de aceptación expresa.
- Registro del consentimiento en tabla `consent`.

   Los datos personales del tutor de la mascota se tratan para:
   - Gestionar la atención veterinaria contratada.

   - TukiFact (RUC [RUC TukiFact]): emisor electrónico de comprobantes.

   - En el local: [dirección]

   Tiempo máximo de respuesta: 20 días hábiles.

8. AUTORIDAD DE CONTROL
   En caso de reclamación, puede acudir a la Autoridad Nacional de
   Protección de Datos Personales (ANPD) del MINJUSDH.

9. SEGURIDAD
   Aplicamos medidas técnicas y organizativas conforme a la Ley 29733
   y su reglamento DS 016-2024-JUS.

10. CAMBIOS

    anticipación.
